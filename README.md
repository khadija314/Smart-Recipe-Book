# 🍽️ Smart Recipe Box

## 📖 Overview

**Smart Recipe Box** is a Java-based desktop application developed as a semester project for the **Object-Oriented Programming (OOP)** course. The application provides a graphical user interface (GUI) that allows users to securely log in, add new recipes, categorize them by cuisine and dessert type, and view saved recipes. The project demonstrates the practical implementation of Object-Oriented Programming concepts, Java Swing, file handling, and exception handling.

---

##  Objectives

The main objectives of this project are:

* Apply Object-Oriented Programming concepts in a real-world application.
* Develop an interactive Graphical User Interface (GUI) using Java Swing.
* Manage recipe data using file handling techniques.
* Implement exception handling for robust and reliable execution.
* Create a simple and user-friendly recipe management system.
